es bajar una copia del repositorio en hithub hacia tu pc

0) ir a github y buscar el repositorio en search con el usuario y nombre del repo, darle al boton clone repository
1) crear directorio local si lo clonará en una unidad obviar paso.
2) abrir git bash
3) cd ruta, moverse al directorio que creo, para ello valla al directorio y copie la ruta,pegar con la tecla insert.
4) git clone url-repositorio  -b develop --single-branch , para clonar el develop branch.
5) cd repositorio local, ir y copiar y pegar aqui.
