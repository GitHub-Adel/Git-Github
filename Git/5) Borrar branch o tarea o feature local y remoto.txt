esto es cuando te acepten el commit de la tarea, porque ya la has terminado bien.
entonces procederemos a eliminarla para empezar con la otra tarea.

1) abrir git bash
2) cd direcorio, posicionarnos en el directorio del repositorio local.
3) git push origin :feature/tarea, borra el branch o reature enviamos al remoto, luego que nos la aprueben.
     este branch debe ser eliminado por el admin al momento de aprobar
4) git branch -D feature/tarea, borra el branch local, nosotros deberemos eliminar del local
5) cerrar issue o nota. por github, para ello ir a Issue y buscar alli.
