estos comando son para ver algunas cosas

1) abrir git bash
2) cd direcorio, posicionarnos en el directorio del repositorio local.
3) git branch, con esto sabremos cuales son las ramas o feature o tareas que tenemos
4) git status, para saber si tenemos cosas pendiente por enviar.
5) git log -2, muestra los ultimos 2 commit,  -p: muestra los detalles del commit.
5) git config credential.helper store,  con esto almaceno las credenciales y asi no me la pide cada vez que quiera hacer commit.
