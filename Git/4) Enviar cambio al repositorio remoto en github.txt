1) me posiciono en el branch p feature que se esta trabajando.
2) git pull origin issue#3,  bajo el repositorio remoto al issue#3, esto trae los cambios de los colaboradores.
3) git add . 
4) git commit -a -m 'descripcion del cambio que has hecho'
5) git push origin issue#3, envio al repositorio remoto mi branch de trabajo issue#3
6) esperar que te acepten el pull request, sino lo aceptan, no seguir el paso 8
7) git branch -D issue#3, borro el branch issue#3
8) git checkout master,  ahora me muevo a mi master local(mi principal local), puede ser develop u otra
9) git pull origin master, luego de integren mis cambios al remoto, bajarlo a mi local para tenerlo al dia.

nota:
---------------
para que entienda esto: 
origin: es el repositorio remoto(NetCoreMVC), mas no una rama.
git pull origin rama : esto baja el repositorio remoto a la rama local especificada.          
git push origin rama: esto sube al repositorio remoto la rama local especificada
