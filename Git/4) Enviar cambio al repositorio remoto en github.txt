es para enviar los trabajo que he hecho hasta ahora.

1) abrir git bash
2) cd  repo, posicionarnos en el directorio del repositorio local.
3) git checkout feature/tarea, posicionarse en la rama o reature a enviar
3) git pull origin master, baja los cambios de los demas colaboradores que no tengo en local.
4) git add . 
5) git commit -a -m 'descripcion del cambio que has hecho'
6) git push origin feature/tarea,  empuja los cambios de tu branch feature/tarea, al repo en github, feature es un        prefijo usado para las tareas no es obligatorio.
