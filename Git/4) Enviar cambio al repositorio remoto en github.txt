es para enviar los trabajo que he hecho hasta ahora. en mi rama feature/tarea y enviarla al remoto.
tambien debe actualizar el master local.

1) abrir git bash
2) cd  repo, posicionarnos en el directorio del repositorio local.
3) git checkout issue#1, posicionarse en la rama o reature a enviar
3) git pull origin master, actualiza mi remoto con el remoto master
4) git add . 
5) git commit -a -m 'descripcion del cambio que has hecho'
6) git push origin issue#1, empuja los cambios de tu issue local a tu remoto
7) git checkout master
8) git pull origin master,  pasa los cambios de mi remoto al master remoto.
