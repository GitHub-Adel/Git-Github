Flujo de trabajo de funcionalidad
---------------------------------------------
Estaremos utilizando este flujo, donde cada colaborador crea una rama apartir de develop
por cada requerimiento que se le asigne.

0) Almacenar credenciales una vez solicitada en local.
1) Crear repositorio remoto (github).
2) Clonar develop  (git).
3) Crear rama desde develop (git).
4) Trabajar rama
5) traer los cambios de develop
6) Empujar rama local al remoto(git).
7) Crear pull request
8) Esperar aprovacion del pull request
9) Cambiar a develop
10) Bajar cambios
11) Eliminar rama trabajada


fin
